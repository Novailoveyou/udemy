# Code repos

Here are the links to all of the projects in the course. If you want to see the code from a specific section, you can view the commits. For the most part, they are in sync with the videos.

## Mini-Project Sandbox

Mini-Project Sandbox: Some of the projects that were geared toward a certain concept were very small, so I put them all in a single repo:

https://github.com/bradtraversy/modern-react-course-sandbox

## Crypto Dash Project

Crypto Dash Project: A dashboard to show coin prices and other information. Filter and sort coins. We also used React Router in declarative mode to show a details page with a chart:

https://github.com/bradtraversy/crypto-dash

## GitHub Finder Project

GitHub Finder Project: Search GitHub users using the public REST API. We used this to demonstrate TanStack Query:

https://github.com/bradtraversy/github-finder-tanstack-query

## The Friendly Dev

The Friendly Dev: A portfolio website that uses React Router v7 in framework mode with SSR enabled and uses Strapi headless CMS on the backend:

Frontend: https://github.com/bradtraversy/friendly-dev-frontend

Strapi Backend: https://github.com/bradtraversy/friendly-dev-backend

## IdeaDrop MERN Project

IdeaDrop MERN Project: A place to share startup ideas and side hustles. It uses TanStack Router and Query on the frontend and a custom Node.js/Express API on the backend.

Frontend: https://github.com/bradtraversy/idea-drop-ui

Backend: https://github.com/bradtraversy/idea-drop-api
